Data Science And Machine Learning Internship

  • The Data Science & Machine Learning Internship provided by UNP (United Network of Professionals) was an excellent introduction to the foundational topics of data science.

  • It initially tackled the basics of Python language, and informed about various aspects like lists, loops, and several packages including pandas, numpy, and seaborn among others.

  • The course futher progressed to introducing datasets into picture, on which we were supposed to apply basic algorithms like linear regression and logistic regression.

  • Further, I was able to learn various Machine Learning algorithms like Decision Trees, Random Forest, Support Vector Machines, K-Nearest Neighbours, etc.

  • We were asked to do a project as a team, and further explain it using a powerpoint presentation in an offline platform.

  • I was able to complete two major machine learning projects - The Used Car Price Prediction and The Bank Term Deposit Prediction.
